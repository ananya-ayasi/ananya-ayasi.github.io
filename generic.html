Speaker Diariazation using Transformers - Ongoing

  • Guided by Dr. Rajeev Rajan
  • Implemented a Bi-LSTM model with self-attention as baseline model and obtained an error rate of 25.7%.
  • Currently experimenting with various deep learning techniques involving transformers to diarize a given audio with data obtained from AMI Speech Corpus
  • A web app using Streamlit will be developed for the same.

Octoechos classification in liturgical music using Deep Learning frameworks - Ongoing

  • Guided by Dr. Rajeev Rajan
  • This automatic classification scheme is addressed using MTF-GRU, MTF-LSTM, MTF-SBU-LSTM, MTF-SB-GRU and MTF-SBU-GRU. The performance of these methods is compared using their accuracy (precision, recall and f1-score).
  • The proposed experiment demonstrates the potential of SBU-LSTM in learning information through MTF.
  • Paper submitted to International Conference on Acoustics, Speech, and Signal Processing 2022.
